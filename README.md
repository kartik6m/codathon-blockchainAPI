# Blockchain based Voting System demo
## Project developed for Codathon, SISW
## Features
- ### From home screen:

    - Login
    - Register

- ### From admin dashboard
	
    - Candidate can be added
    - Vote can be casted once (available candidates shown dynamically)
    - Summary can viewed

- ### From user dashboard

    - Vote can be casted once (available candidates shown dynamically)
    - Summary can viewed

## Technology Used:

- Node.js
- Express middleware
- MongoDB
- HTML Bootstrap
- Postman (for API testing)
